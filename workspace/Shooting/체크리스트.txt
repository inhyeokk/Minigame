연산자, 조건문, 반복문 : 광범위하게 쓰임
클래스, 인스턴스 변수, 메서드 : 광범위하게 쓰임
상속 : 모든 게임 오브젝트가 GameObject에서 공통 멤버와 메서드를 상속받음
다형성 : 게임 오브젝트들의 클래스에서 각각 Rect라는 메서드가 오버로딩되었으며 Me클래스에서는 몇몇 메서드가 오버라이딩 되어짐
제네릭 : 리스트 만들때 다양한 클래스로 형식매개변수를 전달하여 제네릭 사용
인덱서 : 맵 구현을 위하여 맵 정보를 담고 있는 Map과 MapHeight 클래스에 사용
주소 참조 : IsMeet 함수에 out 키워드 사용
인터페이스 : 게임오브젝트들에 다중상속되는 IMove로 구현
예외처리 : 부분부분 IndexOutOfRange 될만한 곳에 삽입함
델리게이터, 람다 : 델리게이터 move함수에 씀.
LINQ : 맨끝에 점수 저장해서 텍스트파일 만드는 부분에 사용
